# React + TypeScript + Vite

Этот проект представляет собой приложение, созданное с использованием React, Redux Toolkit, TypeScript и Vite.

## Установка и запуск

Для установки зависимостей и запуска проекта выполните следующие команды:

```sh
npm install
npm run dev
```
